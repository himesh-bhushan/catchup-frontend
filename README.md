# CatchUp — Health Companion Web Application

CatchUp is a web-based health companion application designed to help users understand and engage with their biometric data in a simple and user-friendly way. The application presents health information clearly and encourages healthier habits through visual feedback and interactive features.

---

## Project Overview

Health tracking devices generate large amounts of data, which can be difficult for users to interpret. CatchUp aims to simplify this information by presenting key health metrics through clean visualisations and a virtual companion, making health data easier to understand and more meaningful.

---

## Key Features

- User account and profile setup  
- Visualisation of health data from trackers  
- Activity ring and health summary dashboard  
- Detailed statistics and historical views  
- Virtual chat assistant for guidance  
- Leaderboard and sharing features  
- Responsive design for mobile, tablet, and desktop  

---

## Technology Stack

- **Frontend:** React.js  
- **Styling:** CSS / Tailwind CSS  
- **Routing:** React Router  
- **Backend:** Node.js with Express (planned)  
- **Database:** MongoDB (planned)  
- **UI/UX Design:** Figma  
- **Version Control:** Git & GitHub
